Detailed Description

This file exposes the API of the P-Sim (or Sim) universal construction. An example of use of this API is provided in benchmarks/simbench.c file.

Author
Nikolaos D. Kallimanis (nkall.nosp@m.ima@.nosp@m.gmail.nosp@m..com)

For a more detailed description see the original publication: Panagiota Fatourou, and Nikolaos D. Kallimanis. "A highly-efficient wait-free universal construction". Proceedings of the twenty-third annual ACM symposium on Parallelism in algorithms and architectures (SPAA), 2011.

This function initializes an instance of the Sim universal construction.

This function should be called once (by a single thread) before any other thread tries to apply any request by using the synchSimStructInit function.

Parameters
sim_structA pointer to an instance of the Sim universal construction.
nthreadsThe number of threads that will use the Sim universal construction.
max_backoffThe maximum value for backoff (usually this is much lower than 100).
