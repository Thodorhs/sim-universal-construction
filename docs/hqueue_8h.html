Detailed Description

This file exposes the API of the H-Queue concurrent queue implementation. An example of use of this API is provided in benchmarks/hqueuebench.c file.

Author
Nikolaos D. Kallimanis (nkall.nosp@m.ima@.nosp@m.gmail.nosp@m..com)

For a more detailed description see the original publication: Panagiota Fatourou, and Nikolaos D. Kallimanis."Revisiting the combining synchronization technique". ACM SIGPLAN Notices. Vol. 47. No. 8. ACM, PPoPP 2012.

◆ synchHQueueInit()

void synchHQueueInit ( SynchHQueueStruct *  queue_object_struct,
uint32_t  nthreads,
uint32_t  numa_nodes 
)

This function initializes an instance of the H-Queue concurrent queue implementation.

This function should be called once (by a single thread) before any other thread tries to apply any enqueue or dequeue operation.

Parameters
queue_object_structA pointer to an instance of the H-Queue concurrent queue implementation.
nthreadsThe number of threads that will use the H-Queue concurrent queue implementation.
numa_nodesThe number of Numa nodes (which may differ with the actual hw numa nodes) that H-Queue should consider. In case that numa_nodes is equal to HSYNCH_DEFAULT_NUMA_POLICY, the number of Numa nodes provided by the HW is used (see more on hsynch.h).
