Detailed Description

This file exposes the API of the SimStack concurrent stack implementation. An example of use of this API is provided in benchmarks/simstackbench.c file.

Author
Nikolaos D. Kallimanis (nkall.nosp@m.ima@.nosp@m.gmail.nosp@m..com)

For a more detailed description see the original publication: Panagiota Fatourou, and Nikolaos D. Kallimanis. "A highly-efficient wait-free universal construction". Proceedings of the twenty-third annual ACM symposium on Parallelism in algorithms and architectures (SPAA), 2011.

struct SynchSimStackThreadState

SynchSimStackThreadState stores each thread's local state for a single instance of SimStack. For each instance of SimStack, a discrete instance of SynchSimStackThreadState should be used.

◆ SynchSimStackStruct

struct SynchSimStackStruct

SynchSimStackStruct stores the state of an instance of the SimStack. SynchSimStackStruct should be initialized using the synchSimStackStructInit function.

Data Fields
ArgVal * announce Pointer to an array, where threads announce only the requests that want to perform to the object.
SimStackState ** pool An array of pools (one pool per thread) of SimStackState structs.
uint32_t nthreads The number of threads that use this instance of SimStack.
int MAX_BACK The maximum backoff value.

void synchSimStackStructInit ( SynchSimStackStruct *  stack,
uint32_t  nthreads,
int  max_backoff 
)

This function initializes an instance of the SimStack concurrent stack implementation.

This function should be called once (by a single thread) before any other thread tries to apply any push or pop operation.

Parameters
stackA pointer to an instance of the SimStack concurrent stack implementation.
nthreadsThe number of threads that will use the SimStack concurrent stack implementation.
max_backoffThe maximum value for backoff (usually this is much lower than 100).
